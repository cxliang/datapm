### 1.1 数据采集工具

| 名称 | 开发公司 | 场景 | 简述 | 优劣性 |
| :--- | :--- | :--- | :--- | :--- |
| Flume | Cloudera | 实时采集日志引擎 | 分布式海量日志采集 |  |
|  |  |  |  |  |



Flume 是一种提供高可用、高可靠、分布式海量日志采集、聚合和传输的系统。它支持在日志系统中定制各类数据进行发送，用于采集数据；同时，它支持对数据进行简单处理，并写到各种数据接收方。
